Deadline Sync - Stay Ahead of Your Deadlines

This app is designed to help you track all the deadlines in your lives. With it's simple UI, keeping track of deadlines have never been easier.

Users are able to add various types of deadlines and the app will keep track of all such deadlines. By implementing local notification system, users will be getting notified when any deadline is about to be met. Firebase is used to host the backend of the app, so data migration is made possible for the users.

These are some possible features that could be implemented in later updates:

  • Repeat Feature.
  • Custom Deadline Categories.
  • Stats
  • UI Improvements.
